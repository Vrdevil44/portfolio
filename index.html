MusicPad

Web-based Music Composition Platform

MusicPad is a collaborative web-based platform that enables users to create, share, and discover original music. Developed in association with the University of the Fraser Valley, this project showcases my skills in web development and audio processing.

The application features real-time audio processing with JavaScript and jQuery, allowing for seamless playback and high-quality sound. Users can create compositions by linking .wav files to keyboard inputs, creating an interactive music creation experience.

Key features include robust exporting capabilities that allow users to share their musical creations across multiple platforms and in various formats, as well as a customizable music planning tool offering users the flexibility to tailor their compositions to specific needs.

Technologies Used

JavaScript jQuery HTML5 CSS3 Web Audio API Git
